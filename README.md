# ğŸ“– [OOP-School-Library] <a name="about-project"></a>

**[OOP-School-Library]()** app is designed to help the librarian of OOP University manage the library's book collection and track the borrowing records of students and teachers. This app provides a well-organized and intuitive interface to perform the following tasks:

- Add new students or teachers to the library system.
- Add new books to the library's collection.
- Maintain records of book borrowing, including information about who borrowed a book and when.
